Auditoria de Estoque

Para esta auditoria, criei uma interface em que é possível obter a quantidade de produtos que estão armazenados em determinado local, registrar os produtos validados, registrar inconformidades e salvar observações, registrando posteriormente toda auditoria em uma planilha.
Todas essas implementações influenciaram positivamente no setor, melhorando a qualidade das auditorias, dando agilidade e podendo registrar informações que antes eram difíceis de se conseguir no momento da auditoria.

Nesses projetos foram utilizados HTML, CSS e o Google Apps Script, linguagem do Google baseada em JavaScript e também é utilizado um banco de dados próprio do KaBuM! disponibilizado no GK-e Manager
